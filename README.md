## 🧩 Overview
ExpenseTracker is a **Python‑based web application** that helps users manage and visualize their daily expenses.  
It provides **category‑wise tracking**, **monthly summaries**, and a clean dashboard built with **Flask, HTML, and CSS**.

---

## 🚀 Features
- 📊 **Expense Dashboard** — view all transactions in one place.  
- 🧾 **Category‑wise Reports** — analyze spending patterns.  
- 💾 **Persistent Storage** — automatically saves data for future sessions.  
- 🔐 **Login System** — secure access for personalized tracking.  
- 🎨 **Responsive UI** — modern design using HTML and CSS.
